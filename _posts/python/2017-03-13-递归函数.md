> python之递归函数

---


## 1、特征
- 是一个函数  
- 函数执行的过程中，自己调用自己
- 一定有结束调用自己的条件
- 效率不高
- 什么问题必须用递归？ 汉诺塔

## 2、实现
~~~ python
# coding=utf-8

def print_sth(n):
    if n < 0:
        return
    print("*: ", n)
    print_sth(n-1)

print_sth(5)

# ret:
"""
*:  5
*:  4
*:  3
*:  2
*:  1
*:  0
"""
~~~
